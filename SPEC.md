---

## 1. 產品概述

### 目標
每天早上全自動生成台股技術分析報告，發布到 GitHub Pages。

### 核心功能
- 爬取加權指數最新數據
- 小金 AI 判斷 3-5 支潛力股（技術分析）
- 生成專業 HTML 報告
- 自動部署到 GitHub Pages

---

## 2. 技術架構

```
GitHub Actions (cron: 08:30 UTC+8)
    ↓
Python Scraper (爬取台股數據)
    ↓
MiniMax API (技術分析生成)
    ↓
HTML Generator (產生日报)
    ↓
GitHub Pages (自動部署)
    ↓
老公打开瀏覽器看報告
```

---

## 3. 數據來源

### 股價數據
- TWSE API: https://mis.twse.com.tw/stock-api/getChart
- 公平價 API: https://api.fairprice.com.tw/

### 技術指標計算
- KD (RSV 9日)
- MACD (EMA 12/26)
- 均線 (MA 5/10/20/60)
